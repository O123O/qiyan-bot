# WeChat adapter

Status: Planned

WeChat support is not implemented in this release. A future adapter must map its message and attachment APIs into the same normalized backend contracts while preserving the single-owner authorization and durable-delivery rules.

There is currently no supported WeChat account type, callback configuration, credential configuration, or installation procedure. Do not connect a WeChat account to this binary expecting it to work. Track the repository's future releases for an implemented adapter and a platform-specific setup guide.
